二分法求函数的零点

二分法求函数的零点

题目来源：

openjudge
NOI(题库正在建设中，做题纪录有可能会被删除，请注意) / 1.11编程基础之二分查找 第二题
提交链接：openjudge

AC代码
#include<cstdio>
#include<cstring>
#include<algorithm>
using namespace std;
double l=1.5,r=2.4;
inline int ok(double x)
{
    double ans=0;
    ans=x;
    ans-=15;
    ans*=x;
    ans+=85;
    ans*=x;
    ans-=225;
    ans*=x;
    ans+=274;
    ans*=x;
    ans-=121;
    if(ans>0)return 1;
    return 0;
}
int main()
{
    while((r-l)>0.000001)
    {
        double mid=(l+r)/2.0;
        if(ok(mid))
        {
            l=mid;
            continue; 
        }
        r=mid;
    }
    printf("%.6lf",r);
}
总结

这个题目基本没有什么坑点，唯一坑点就是对于精度的把握和四舍五入。
精度方面他让你到小数点后x位，while里面写(r-l)<10^(-x)就可以了，写少了不行，但是写多了不影响。
关于四舍五入，直接输出r就可以了，一般r截掉后面几位之后就是四舍五入的结果（现在想想好像不太对…）
